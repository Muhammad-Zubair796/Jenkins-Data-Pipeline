# Automated Data Engineering Pipeline
This project demonstrates a production-ready CI/CD pipeline using **Jenkins** and **Docker**.

### Features:
- **Automated SCM:** Triggered from GitHub.
- **Containerized Execution:** Runs inside a Python 3.9 Docker container for environment isolation.
- **Data Validation:** Python script includes logic to ensure data quality.
- **Infrastructure as Code:** Defined via a `Jenkinsfile`.

### Tools Used:
- **Jenkins** (Orchestration)
- **Docker** (Containerization)
- **Python** (Data Transformation)
- **AWS EC2** (Cloud Hosting)
